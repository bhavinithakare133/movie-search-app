# 🎬 CineSearch – Movie Search App

A responsive, dark-themed movie search application built with vanilla JavaScript and the OMDB API.

## 🚀 Live Demo
> Deploy on GitHub Pages or Vercel and add your link here

## 📸 Features
- 🔍 Search movies, series, and episodes in real-time
- 🎭 Filter by type (Movie / Series / Episode)
- 📋 Movie detail modal with plot, cast, director, IMDB rating
- ⚡ Loading states and error handling
- 📱 Fully responsive on mobile and desktop
- 🎨 Dark cinematic UI with smooth animations

## 🛠️ Tech Stack
- HTML5, CSS3, Vanilla JavaScript (ES6+)
- OMDB REST API integration
- CSS Grid & Flexbox for responsive layout
- CSS Animations & Transitions
- Fetch API for async data fetching

## 📦 How to Run
```bash
# Clone the repo
git clone https://github.com/bhavinithakare133/movie-search-app.git

# Open in browser
open index.html
# OR use Live Server in VS Code
```

## 🔑 API
Uses the free [OMDB API](http://www.omdbapi.com/) for movie data.
